    How to use these insights

    • Prioritize previous acceptors first, then strengthen selection using spending, purchasing activity and recency.
    • Use age and income as supporting segment signals, not as the only targeting rule.
    • Test the priority audience against a control group before scaling the next campaign.
    • These are historical propensity insights—not guaranteed predictions or a validated machine-learning model.
